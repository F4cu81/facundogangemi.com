- sophisticated

# Typography Rules

Prioritize:
- whitespace
- readability
- clean hierarchy
- concise paragraphs

Avoid:
- oversized startup typography
- decorative fonts
- dense paragraphs
- excessive bold usage

---

# Text Density

The website should:
- avoid long text blocks
- prioritize scanning
- communicate ideas quickly

Preferred:
- short paragraphs
- strategic spacing
- visual breathing room
