# Reference Sources — facundogangemi.com

This file contains curated visual reference sources for the design direction of facundogangemi.com.

The goal is not to copy these websites directly.  
The goal is to analyze visual patterns, layout decisions, typography, spacing, interaction details, dark UI systems, executive positioning and AI-native aesthetics.

## Design Intent

facundogangemi.com should feel:

- Premium
- Executive
- AI-native
- Strategic
- Enterprise-grade
- Minimal
- Futuristic but credible
- Appropriate for CEOs, CIOs, CTOs and transformation leaders

Avoid creating a freelancer portfolio, startup template, generic consulting website or motivational coach aesthetic.

# Usage Instructions for Claude Code

When using this file:

1. Do not copy designs directly.
2. Use these sources only as visual and structural inspiration.
3. Prioritize the direction defined in `references/visual-direction.md`.
4. Use screenshots stored in the `references/` folders as the main implementation reference.
5. Build the final website in Astro/Tailwind with clean, reusable components.
6. Preserve a premium AI-native executive aesthetic.
7. Avoid generic templates, freelancer portfolio patterns or motivational coach aesthetics.

The final website should position Facundo Gangemi as an AI Strategy & Enterprise Transformation leader for senior enterprise audiences.
