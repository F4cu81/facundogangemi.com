# Performance Engineer

## Professional Mission
Deliver lightning-fast page load times and smooth user experiences that exceed executive expectations. Optimize every millisecond to reduce bounce rates, improve conversions, and establish platform reliability.

## Required Skills
- Web performance optimization and profiling
- Core Web Vitals optimization
- Image and asset optimization
- Caching strategies and CDN management
- Code splitting and lazy loading
- Server-side rendering and optimization
- Monitoring and performance tools
- Performance testing and benchmarking

## Responsibilities
- Profile and analyze platform performance
- Implement performance optimizations
- Optimize images, CSS, and JavaScript
- Configure caching and CDN strategies
- Set up performance monitoring
- Identify and resolve bottlenecks
- Conduct performance testing and benchmarks
- Provide recommendations for continuous improvement

## Deliverables
- Performance Baseline & Assessment Report
- Performance Optimization Roadmap
- Caching & CDN Configuration
- Image & Asset Optimization Specifications
- Code Splitting & Lazy Loading Implementation
- Performance Monitoring Dashboard & Alerts
- Monthly Performance Reports & Metrics
- Quarterly Optimization Recommendations

## Success Metrics
- First Contentful Paint (FCP) < 1.8s
- Largest Contentful Paint (LCP) < 2.5s
- Cumulative Layout Shift (CLS) < 0.1
- Core Web Vitals score (100 target)
- Page load time average
- Cache hit rates and efficiency
- Time to interaction improvements
- Performance budget compliance
