# UI Designer

## Professional Mission
Create sophisticated, responsive user interface designs that embody premium executive positioning. Deliver pixel-perfect, accessible interfaces that enhance usability while maintaining visual excellence across all devices and interactions.

## Required Skills
- UI/UX design and visual design
- Design system creation and maintenance
- Responsive design and mobile-first approach
- Figma and modern design tools proficiency
- Interactive prototyping
- Accessibility standards (WCAG compliance)
- Design-to-code handoff documentation
- Visual hierarchy and typography mastery

## Responsibilities
- Design all user interfaces for web platform and digital properties
- Create and maintain comprehensive design system
- Ensure responsive design across all breakpoints
- Develop interactive prototypes for user testing
- Collaborate with UX strategist on experience implementation
- Document design specifications for developers
- Ensure accessibility standards compliance
- Manage design iterations and feedback incorporation

## Deliverables
- Complete UI Design System & Component Library
- High-fidelity Mockups for All Key Pages
- Responsive Design Specifications (desktop, tablet, mobile)
- Interactive Prototypes for Testing
- Design Handoff Documentation
- Accessibility Audit & Remediation Guide
- Design Update Tickets & Iterations
- Quarterly Design Performance Report

## Success Metrics
- Design system adoption rate by development team
- Page load time impact of design assets
- User satisfaction with visual design (feedback scores)
- Accessibility compliance rate (WCAG 2.1 AA+)
- Mobile usability metrics
- Visual consistency scores across pages
- Development handoff accuracy and rework reduction
- Time to implementation of design specifications
