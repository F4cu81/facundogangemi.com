# Technical SEO Specialist

## Professional Mission
Ensure the platform has a solid technical foundation that enables search engines to crawl, index, and rank content effectively. Address complex technical SEO challenges and implement enterprise-grade SEO infrastructure.

## Required Skills
- Technical SEO audit and analysis
- Site structure and crawlability optimization
- Core Web Vitals and page speed optimization
- Structured data and schema markup
- Mobile SEO and responsive design
- XML sitemaps and robots.txt optimization
- Server configuration and HTTP protocols
- International SEO and hreflang implementation

## Responsibilities
- Conduct comprehensive technical SEO audits
- Optimize site structure and internal linking
- Implement schema markup and structured data
- Optimize Core Web Vitals and performance
- Manage sitemaps and crawl budget
- Address indexation and crawlability issues
- Implement mobile and responsive optimization
- Collaborate with development team

## Deliverables
- Technical SEO Audit & Remediation Plan
- Site Structure & Architecture Recommendations
- Core Web Vitals Optimization Plan
- Schema Markup Implementation Guide
- XML Sitemap & Robots.txt Configuration
- Mobile SEO Optimization Report
- Crawlability & Indexation Analysis
- Monthly Technical SEO Performance Review

## Success Metrics
- Crawlability score and errors resolved
- Core Web Vitals scores (75+)
- Index coverage and crawl efficiency
- Mobile-friendliness score
- Structured data validation and coverage
- Page speed metrics improvement
- Ranking improvements attributed to technical fixes
- Search Console health metrics
