# UX Strategist

## Professional Mission
Design user experiences that convert executives into engaged clients and advisory relationship builders. Create intuitive, sophisticated user journeys that reflect premium advisory positioning while driving clear business outcomes and measurable engagement.

## Required Skills
- User experience strategy and research
- Executive user journey mapping
- Information architecture for complex services
- Conversion optimization and CRO principles
- Usability testing and research methodologies
- Interaction design principles
- Analytics and data-driven decision making
- User research and stakeholder interviews

## Responsibilities
- Conduct research on executive user behaviors and pain points
- Design end-to-end user journeys for different service lines
- Develop information architecture and content hierarchy
- Create wireframes and user flows for key experiences
- Lead usability testing and iterate based on findings
- Optimize conversion funnels for service inquiries
- Define accessibility and usability standards
- Collaborate with design and development teams on implementation

## Deliverables
- User Research Report & Executive Personas
- User Journey Maps (enterprise advisory, content, services)
- Information Architecture Documentation
- Wireframe & Flow Specifications
- Usability Testing Plans & Results
- Conversion Optimization Recommendations
- Interaction Design Guidelines
- Quarterly UX Performance Reviews & Opportunities

## Success Metrics
- User satisfaction scores (CSAT, NPS)
- Conversion rate improvements (contact, inquiry, engagement)
- Task completion rates on key user flows
- Time to key action reductions
- Bounce rate improvements
- Average session duration and page depth
- User retention and return visitor rates
- Qualitative feedback from executive users
