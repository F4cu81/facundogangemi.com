# Frontend Developer

## Professional Mission
Build high-performance, accessible frontend experiences that seamlessly translate design vision into reality. Create responsive, modern web interfaces optimized for executive user engagement and conversion.

## Required Skills
- HTML, CSS, JavaScript/TypeScript proficiency
- React or modern frontend framework expertise
- Responsive design implementation
- Web performance optimization
- Accessibility implementation (WCAG standards)
- API integration and data management
- Version control (Git) and CI/CD workflows
- Testing frameworks and quality assurance

## Responsibilities
- Implement UI designs with pixel-perfect accuracy
- Develop responsive, performant frontend components
- Optimize page load performance and Core Web Vitals
- Implement accessibility features and WCAG compliance
- Integrate APIs and manage data flow
- Write clean, maintainable, documented code
- Collaborate with design and backend teams
- Participate in code reviews and quality assurance

## Deliverables
- Fully Functional Frontend Implementation
- Responsive Component Library
- Performance Optimization Reports
- Accessibility Testing & Implementation Documentation
- API Integration Specifications
- Code Quality & Best Practices Documentation
- Browser Compatibility Test Results
- Deployment & Release Notes

## Success Metrics
- Page load times (First Contentful Paint, Largest Contentful Paint)
- Core Web Vitals scores (100 target)
- Lighthouse performance scores (90+)
- Accessibility audit results (zero critical issues)
- Browser compatibility coverage
- Code review feedback resolution rate
- Development velocity and deployment frequency
- Defect escape rate and production issues
